Detailed Description

template<typename T, typename Combine>
class fenwick< T, Combine >

Fenwick Tree that answers prefix queries in O(log N) with very low constant factor.

The difference with Segment Tree is: 1) Cannot directly set an element, instead update it through the combine function. 2) Very low constant factor for its operations.
