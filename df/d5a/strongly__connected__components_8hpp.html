vector< vector< int > > strongly_connected_components ( vector< vector< int > > &  g)

Find all strongly connected components (SCCs) in a directed graph.

A strongly connected component is a maximal subgraph where every vertex is reachable from every other vertex. Uses Kosaraju's algorithm with two DFS passes to identify all SCCs.

Time Complexity: \(O(V + E)\) where V is vertices and E is edges. Space Complexity: \(O(V + E)\) for storing the transpose graph.

Parameters
gThe adjacency list representation of the directed graph.
Returns
A vector of vectors, where each inner vector contains the vertices of one strongly connected component.
Note
The SCCs are returned in reverse topological order of the condensation graph.
