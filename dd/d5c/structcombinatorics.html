Detailed Description

template<int M>
struct combinatorics< M >

Precomputes factorials and inverses for fast combinatorial calculations.

Enables O(1) computation of binomial coefficients and permutations using precomputed factorials, inverse factorials, and modular inverses. Uses Fermat's little theorem for computing modular inverses, requiring prime modulus.

Time Complexity:

  • Construction: O(N)
  • ncr(), npr(): O(1) Space Complexity: O(N)
Template Parameters
MThe modulus (must be prime).
Note
Modulus must be prime for Fermat's little theorem to work.
Maximum n must be known at construction time.
