Detailed Description

template<typename T, typename Combine>
class segment_tree< T, Combine >

Iterative Segment Tree for point updates and range queries.

A versatile data structure supporting efficient range queries and point updates for any associative operation. This iterative implementation is faster and simpler than recursive versions, with excellent cache locality.

Template Parameters
TValue type.
CombineBinary associative operation (e.g., min, max, sum, gcd).
Note
Combine operation must be associative.
Supports efficient binary search on prefixes/suffixes.
