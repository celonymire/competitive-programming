Detailed Description

template<typename T, typename Combine>
class heavy_light_decomposition< T, Combine >

Heavy-Light Decomposition for path queries on trees.

Decomposes a tree into heavy and light edges, allowing path queries and updates to be answered using a segment tree. Each root-to-node path crosses at most \(O(\log N)\) light edges, enabling efficient queries.

Template Parameters
TValue type.
CombineBinary associative operation for combining path values.
Note
Each path crosses at most log N chains.
For edge-weighted trees, store edge weights at the deeper endpoint.
