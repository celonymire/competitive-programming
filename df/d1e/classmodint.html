Detailed Description

template<int M>
class modint< M >

Keeps the internal 32-bit integer value modulo M.

The modulo is taken as a constant template parameter to optimize for the common case of fixed modulo, and it assumes the modulus is prime.

Template Parameters
MThe modulus.
