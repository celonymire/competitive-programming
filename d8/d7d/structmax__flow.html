Detailed Description

template<typename F>
struct max_flow< F >

Dinic's algorithm for maximum flow.

Computes the maximum flow in a directed graph with given capacities in \(O(VE^2)\) time with \(O(VE)\) space.

Template Parameters
FFlow capacity type.
See also
https://codeforces.com/blog/entry/66006 for scaling optimization.
