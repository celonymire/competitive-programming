Detailed Description

Cryptographically-inspired hash function resistant to collision attacks.

Standard unordered_map hash functions are vulnerable to adversarial input that causes many collisions, degrading performance to O(N) per operation. This hasher uses a random seed and mixing functions to prevent such attacks.

Time Complexity: O(1) per hash Space Complexity: O(1)

Note
Essential for contests where adversaries can craft anti-hash test cases.
Uses a random seed initialized at program startup.
